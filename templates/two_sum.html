Two Sum

Given an array of integers nums and a target, return indices of the two numbers such that they add up to target.

Example

Input: 2 7 11 15 9 Output: 0 1

Hints

Think about storing numbers in a HashMap.
Check if target - current number already exists.

Solution

Please attempt the problem once before viewing the solution.
Optimized Python Solution (O(n)) 
nums = list(map(int,input().split()))
target = int(input())

seen = {}

for i in range(len(nums)):
    complement = target - nums[i]

    if complement in seen:
        print(seen[complement], i)
        break

    seen[nums[i]] = i
Time Complexity: O(n)
